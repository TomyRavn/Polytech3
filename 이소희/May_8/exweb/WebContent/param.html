파라미터를 포함한 요청을 보내는 방법

1.웹브라우저의 주소창에 주소와 파라미터를 직접 입력 (GET 방식 요청)

http://컴퓨터 주소:포트 번호/요청 파일 경로?파라미터 이름=파라미터 값&파라미터 이름 =파라미터값

2.링크를 클릭하여 href 속성의 주소로 요청 전송(GET 방식 요청)

HelloServlet이 링크를 클릭하면 이동 HelloServlet이 링크를 클릭하면 이동 HelloServlet이 링크를 클릭하면 이동 //컴퓨터 주소나 폴더 명이 같은 부분은 생략이 가능하다, but 파라미터 값이 같을 때만 가능!

3. form 내부의 submit 버튼을 클릭하여 action 속성의 주소로 요청 전송(파라미터 값을 변경 가능하다!)

어떤 주소로 요청을 보낼 때, 다양한 요청 방식(== HTTP메서드 or ==HTTP 커맨드) 존재 : GET, POST, PUT, DELETE, ...
form 엘리먼트의 method 속성으로 요청 방식 지정 가능
